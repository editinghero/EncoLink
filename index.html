Help & FAQ

How It Works

EncoLink uses AES-256 encryption to protect your URLs. When you encrypt a link, it creates a secure URL that can only be accessed with the correct password.

Security

All encryption happens in your browser. No passwords or URLs are sent to any server. Your data remains completely private.

Time-Lock Security

Time-locked links use verified server time to prevent bypassing by changing device time. The system automatically detects your timezone and validates access using atomic clock APIs. An internet connection is required to verify time-locked links.

Clipboard

Use the clipboard feature to quickly detect and encrypt multiple URLs at once. Simply copy text containing URLs and let EncoLink find them automatically.
